# Sistema de Administración de Citas Clínicas

Esta aplicación fue desarrollada en Java con el objetivo de simular un sistema de administración de citas para un consultorio clínico.

El sistema permite:
- Dar de alta doctores.
- Dar de alta pacientes.
- Crear citas con fecha, hora y motivo.
- Relacionar cada cita con un doctor y un paciente.
- Control de acceso mediante administrador.

Toda la información es almacenada en archivos CSV dentro de la carpeta db, garantizando portabilidad del sistema.

La aplicación fue diseñada bajo los principios de Programación Orientada a Objetos, aplicando conceptos como herencia, encapsulamiento, separación de responsabilidades y uso del Collection Framework.
